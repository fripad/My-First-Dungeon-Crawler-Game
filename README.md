# My First Dungeon Crawler (C# Console Application)

## Description
A simplified dungeon crawler built as a C# console application.  
The player explores a predefined dungeon, encounters enemies, and engages in turn-based combat inspired by classic roguelike games.

- C#
- .NET (Console Application)

## Features
- Navigate through a predefined dungeon
- Turn-based combat system
- Enemy encounters with basic AI
- Player health and combat mechanics
